# Requirements

## 1. Project Overview

poleinlove는 폴댄스 기술 기록과 회고를 위한 개인 아카이브 서비스다.
과거 운동 기록을 다시 검색하고, 어떤 점이 어려웠는지와 어떤 방식이 도움이 되었는지 되짚어볼 수 있도록 돕는다.

## 2. Problem

운동 기록이 단순히 쌓이기만 하면 나중에 다시 참고하기 어렵다.
특정 기술을 오랜만에 다시 연습할 때, 예전에 어떤 점이 어려웠고 어떤 팁이 있었는지 빠르게 찾을 수 있는 구조가 필요했다.

## 3. Target User

- 1차 사용자: 나
- 폴댄스 기록을 남기고, 나중에 기술별로 다시 찾아보려는 사람

## 4. MVP Scope

### Included

- 기록 작성
- 기록 목록 조회
- 기록 상세 조회
- 기술명/키워드 검색
- 태그 선택 및 필터
- 사진 업로드

### Excluded

- 달력 뷰
- 통계 기능
- 소셜 기능
- AI 자동 요약
- 복잡한 인증 기능

## 5. User Scenarios

### Scenario 1. 기록 작성

사용자는 운동 후 기록 작성 페이지에서 날짜, 기술명, 태그, 사진, 회고 내용을 입력하고 저장한다.

### Scenario 2. 기술 검색

사용자는 특정 기술명을 검색해 과거 기록을 조회하고, 당시 어려웠던 점과 개선 포인트를 확인한다.

### Scenario 3. 기록 회고

사용자는 기록 상세 페이지에서 하나의 기록을 열어보고, 같은 기술의 다른 기록도 함께 탐색한다.
